# Arduino-Uno-RGB-Strip-Lights

Control RGB LED Strip Lights with an Arduino!  
  
## Supplies  
* 1x Arduino Uno
* 1x Potentiometer
* 1x RGB LED Strip
* 1x 200 Ohm Resistor
* 3x PN2222
* 1x 4 Male Pin Header
* 1x Push Button
